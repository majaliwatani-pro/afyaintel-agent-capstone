# AfyaIntel Portfolio Website

## Open locally

Double-click `index.html`, or run:

```powershell
Set-Location -Path "C:\Users\cohema\agy2-projects\kaggle-5day-ai-agents-2026\portfolio"
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Publish with GitHub Pages

1. Push the repository to GitHub without `.env` or secrets.
2. In repository settings, enable Pages.
3. Choose a deployment source that serves the `portfolio` folder, or copy the portfolio into a dedicated Pages branch.
4. Test all relative links after publishing.

## Before publishing

- Add only contact links you are comfortable making public.
- Confirm no API key, token, personal address, or private screenshot is present.
- Keep the prototype and responsible-use disclaimer visible.
