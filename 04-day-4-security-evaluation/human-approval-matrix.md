# Human Approval Matrix

| Operation | Agent may draft? | Agent may execute? | Required human role |
|---|---:|---:|---|
| Inventory summary | Yes | Yes, read-only | None for viewing |
| Low-stock alert | Yes | Yes, read-only | None for viewing |
| Expiry alert | Yes | Yes, read-only | None for viewing |
| Weekly report | Yes | Yes, draft only | Facility manager reviews |
| Replenishment recommendation | Yes | No | Facility manager / procurement officer |
| Stock redistribution | Yes | No | Authorized store or district officer |
| Clinical diagnosis | No | No | Qualified clinician |
| Prescription or dosage | No | No | Qualified clinician |
| Supplier payment | No | No | Authorized finance officer |
| External message | Draft only | No | Authorized staff member |
