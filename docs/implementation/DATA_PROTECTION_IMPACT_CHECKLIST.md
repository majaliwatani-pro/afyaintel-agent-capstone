# Data Protection Impact Checklist

AfyaIntel must be treated as a health data system if real facility records are introduced.

## Before real data

- [ ] Confirm lawful basis and facility authorization.
- [ ] Document data categories and retention period.
- [ ] Minimize patient-identifiable data.
- [ ] Use role-based access control.
- [ ] Encrypt storage and synchronization channels.
- [ ] Keep audit logs without raw patient text where possible.
- [ ] Establish human clinical review responsibilities.
- [ ] Prepare incident-response and breach-notification process.
- [ ] Validate interoperability exports before external exchange.
- [ ] Train users on clinical boundary and prohibited uses.

## Prototype rule

Use synthetic or de-identified data only.
